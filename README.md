# Proyecto Final React - Rick and Morty

Este repositorio contiene el proyecto final creado con React (usando Vite). La aplicación consume la API pública de Rick and Morty para mostrar los personajes de la serie e incluye un sistema para gestionarlos.

## Funcionalidades implementadas

- Listado de personajes
- Filtro de búsqueda
- Paginacion
- Vista especifica
- Favoritos 

## Tecnologias utilizadas

- React (Vite)
- Yarn (Gestor de paquetes)
- React Router DOM (Manejo de rutas)
- CSS Modules (Estilos independientes por componente)
